# Bridge
Inclusive learning powered by AI.

## Project Structure
- **frontend/**: React + Vite application (UI)
- **backend/**: Node.js + Express application (API)

## Getting Started

### Prerequisites
- Node.js installed

### 1. Start the Backend
Open a terminal and run:
```bash
cd backend
npm install
npm start
```
The backend will run on `http://localhost:5000`.

### 2. Start the Frontend
Open a **new** terminal window and run:
```bash
cd frontend
npm install
npm run dev
```
The frontend will run on `http://localhost:5173` (check the terminal for the exact URL).

## Testing
1. Open the frontend URL in your browser.
2. Paste some text into the text area.
3. Click **Summarize** to see the AI summary (currently mocked).
4. Click **Read Aloud** to hear the text.
